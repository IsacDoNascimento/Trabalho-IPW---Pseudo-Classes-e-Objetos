01. O que são Pseudo-classes?

Pseudo-classes são utilizadas para adicionar efeitos especiais a alguns seletores. Elas funcionam como se uma classe fosse adicionada a uma tag de modo automático, não sendo necessário ao programador ter que declarar o atributo class ao elemento.


Sintaxe:

selector:pseudo-class {
    property: value; 
}

Exemplo (first-child): Simula como se fosse criada uma classe "primeiro-filho" anexada à tag que representa o primeiro filho de determinado elemento.


li:first-child {
    font-weight: bold;
}

02. Pseudo-classes para Links.

Links podem ser configurados quanto aos seguintes estados: Link ativo, Link visitado, Link selecionado e Link mouseover.


/* Link não visitado */
a:link { color: #FF0000; }

/* Link visitado */
a:visited { color: #00FF00; }

/* Mouse sobre o link */
a:hover { color: #FF00FF; }

/* Link selecionado / ativo */
a:active { color: #0000FF; }

04. Pseudo-elementos.

De modo semelhante às pseudo-classes, os pseudo-elementos funcionam como se fosse adicionada uma nova tag (elemento) à página web de modo automático.


Sintaxe:

selector::pseudo-element {
    property: value;
}

Problema prático: Tornar a primeira letra de um parágrafo maior sem alterar o HTML. Solução com ::first-letter:


h1::first-letter {
    font-size: 250%;
}

06. Conclusão e Perguntas.

O domínio de pseudo-classes e pseudo-elementos reduz drasticamente a necessidade de classes extras no HTML e scripts de JavaScript para efeitos visuais simples, tornando a folha de estilo muito mais poderosa.
